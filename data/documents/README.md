# База знаний курса

Положите сюда документы для RAG:

| Файл | Описание |
|------|----------|
| `FAQ.pdf` | Частые вопросы |
| `course.txt` | Материалы курса (демо-файл уже в репозитории) |
| `RTFM.docx` | Ошибки при работе с Cursor |

Также можно загружать документы через веб-интерфейс (кнопка «Добавить в базу знаний»).

Поддерживаемые форматы: PDF, TXT, MD, DOCX.
