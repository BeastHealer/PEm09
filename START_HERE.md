# 🚀 START HERE — Ассистент по промпт-инжинирингу

## О проекте

**Мультимодальный личный ассистент** для студентов курса по промпт-инжинирингу.

Гибридный помощник, который совмещает:
- 📚 **Базу знаний** — ответы из материалов курса (RAG)
- 🎤 **Голос** — распознавание (Whisper) и синтез речи (TTS)
- 📸 **Анализ изображений** — скриншоты ошибок, код, интерфейс Cursor
- 💬 **Контекст диалога** — помнит историю переписки

### Интерфейсы
1. **Веб-сайт** (Flask) — основной, лаконичный современный UI
2. **Telegram-бот** — дублирующий интерфейс с теми же возможностями

### База знаний (`data/documents/`)
| Файл | Содержание |
|------|------------|
| `FAQ.pdf` | Частые вопросы |
| `course.txt` | Материалы курса |
| `RTFM.docx` | Ошибки при работе с Cursor |

### Для кого
Студенты курса — помощь с домашними заданиями, методическими и техническими вопросами.

## ⚡ Быстрый старт

### 1. Ключи API

**ProxyAPI** (для всех AI-функций):
1. [proxyapi.ru](https://proxyapi.ru) → регистрация
2. Раздел **Ключи API** → создать ключ
3. Пополнить баланс

**Telegram** (для бота):
1. [@BotFather](https://t.me/BotFather) → `/newbot`
2. Скопировать токен

### 2. Установка

```bash
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
```

Установите **FFmpeg** для голосовых сообщений.

### 3. Настройка `.env`

```env
PROXYAPI_KEY=ваш_ключ_proxyapi
TELEGRAM_BOT_TOKEN=ваш_токен_бота
SECRET_KEY=случайная_строка
BOT_MODE=rag
```

### 4. Запуск

```bash
python main.py
```

- **Веб:** http://127.0.0.1:5000
- **Telegram:** найдите бота и отправьте `/start`

Если `TELEGRAM_BOT_TOKEN` не задан — запустится только веб-интерфейс.

Если Telegram недоступен (таймаут, блокировка сети) — веб продолжит работать.
Чтобы отключить Telegram явно: `TELEGRAM_ENABLED=false` в `.env`.

## Использование

### Веб-интерфейс
- Напишите вопрос о курсе или домашнем задании
- 📷 — загрузите скриншот (ошибка Cursor, код)
- 🎤 — запишите голосовое сообщение
- Режим **RAG** (по умолчанию) — ответы из базы знаний

### Telegram-бот
```
/start   — начать
/help    — справка
/mode    — режим (rag / voice / vision / text)
/reset   — очистить историю
/stats   — статистика базы знаний
```

Отправьте текст, голосовое или фото — бот ответит с учётом контекста.

---

## Режимы

| Режим | Описание |
|-------|----------|
| `rag` | Ответы из FAQ, course.txt, RTFM.docx (по умолчанию) |
| `text` | Диалог с ассистентом курса |
| `voice` | Ответы озвучиваются (TTS) |
| `vision` | Фокус на анализе изображений |
